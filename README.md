# Stargate Website Project

This is a personel project to build a static website that contains important information about Stargate sci-fi universe.

## Description

The site contains information about Stargate technologies, races, politics, lore and my personel thoughts.

Built with with vanilla HTML, CSS and Javascript.

## How to use

Open 'Index.html' file with a browser to view the website.

No build tools dependencies are required.
